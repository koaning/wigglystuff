# Wiggly notebooks, zero friction


"A collection of creative AnyWidgets for Python notebook environments."


## Install wigglystuff

 uvpip

```
uv pip install wigglystuff
```


```
pip install wigglystuff
```


## What you can build

  Slider2D DemoAPI  Matrix DemoAPI  Paint DemoAPI  ThreeWidget DemoAPI  EdgeDraw DemoAPI  SortableList DemoAPI  ColorPicker DemoAPI  GamepadWidget DemoAPI  KeystrokeWidget DemoAPI  SpeechToText DemoAPI  CopyToClipboard DemoAPI  CellTour DemoAPI  WebcamCapture DemoAPI  ImageRefreshWidget DemoAPI  HTMLRefreshWidget DemoAPI  ProgressBar DemoAPI  PulsarChart DemoAPI  TextCompare DemoAPI  EnvConfig DemoAPI  Tangle DemoAPI

Each widget page embeds a marimo-powered html-wasm export and links back to the exact notebook that generated the demo, so you can open the original `.py` file and rerun it locally.
