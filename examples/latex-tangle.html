Python API behavior, prototyped in HTML

One formula, many parameters.

Drag any colored number or symbol. Repeated symbolic parameters reveal all their numeric occurrences during the gesture, then return to notation when released.

1. Numeric parameters

Both values are controls. Each parameter has independent bounds, steps, and colors.

2. Symbolic parameters

Drag either occurrence of a: both become numbers while b stays symbolic.

3. Mixed presentation

A numeric amplitude and symbolic decay rate coexist in one formula.

Dragging is clamped and snapped to the stated step. Press Enter to commit typed input; Escape or clicking away cancels it. The theme switch previews per-parameter {light, dark} colors. KaTeX lays out each target via \htmlData.
